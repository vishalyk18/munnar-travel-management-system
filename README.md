 Munnar Travel Management System

A full-stack web application developed to manage  customer bookings and travel operations efficiently. This system provides secure authentication, user-friendly interfaces and seamless backend integration for real-time data handling.


 Features

*  JWT-based Authentication and Authorization
*  User & Admin Role Management
*  Booking System
*  CRUD Operations for Users.
*  RESTful API Integration
*  Responsive Frontend Design
* Input Validation and Exception Handling

##  Architecture

The application follows a layered architecture:

* Controller Layer → Handles HTTP requests
* Service Layer → Contains business logic
* Repository Layer → Database interaction



##  Application Flow

1. User/Admin logs in using credentials
2. JWT token is generated and used for authentication
3. User can view tour packages and make bookings
4. Admin can manage packages and user data
5. All operations interact with backend APIs and database
